template<typename T>
class labGrapheAlgorithmes::Graphe< T >

Classe générique représentant un graphe.

La classe gère un graphe générique. L'implémentation se fait par des listes d'adjacence. Les numéros de sommets débutent à 0. Le type T correspond aux noms des sommets, qui peuvent être optionnellement intégrés.
